使用指南
  1. 拖动右侧元件图标到左侧即可放置
  2. 右栏可编辑元件属性，支持格内的水平垂直微调
  3. 按住 Shift 点击可选中多个元件并组合
  4. 顶栏可调渲染精细度
  5. 顶栏可隐藏元件、切换主题、导出图片
  6. 右栏可导出当前配置，下次加载配置即可复原光路
  7. 快捷键：R 旋转 Del 删除 Esc 取消 Ctrl+C/V 复制/粘贴
  8. 详情界面见 github.com/Charm-Lord/OpticalBoard

How to use
  1. Drag an element icon from the right onto the canvas to place it
  2. Edit element properties on the right, with fine in-cell X / Y nudging
  3. Hold Shift and click to select several elements and group them
  4. Adjust render quality from the top bar
  5. Hide elements, switch theme, and export images from the top bar
  6. Export the current config on the right, load it later to restore the setup
  7. Keys: R rotate Del delete Esc cancel Ctrl+C/V copy/paste
  8. Full details at github.com/Charm-Lord/OpticalBoard
